# Comando `init` - Criação Interativa de Manifest

Execute `sagep-auth-cli init` para criar um manifest YAML interativamente. O wizard guia você passo a passo: informações da aplicação, usuários (Master e comuns), permissões e roles base. O manifest é salvo em `./auth-manifest.yaml` (ou caminho especificado via `--manifest`). Execute `sync` depois para sincronizar com o servidor.
